/* * sakura.css - ミニマルなカスタマイズひな形 * H1, H2, P のフォントサイズのみ指定 */ /* ================================================================= * 1. 基本設定 * ================================================================= */ /* H1の見出しサイズを指定 */ h1 { /* 例: 画面全体を占めるような、非常に大きな見出し */ font-size: 3rem; /* 3rem はデフォルトのフォントサイズ（通常16px）の3倍です */ } /* H2の見出しサイズを指定 */ h2 { /* 例: セクションのタイトルに使う、H1よりは控えめなサイズ */ font-size: 2rem; } /* P（段落）の文字サイズを指定 */ p { /* 例: 読みやすいように、デフォルトより少し大きめのサイズ */ font-size: 1.1rem; } /* ================================================================= * 2. （参考）その他の要素のサイズを調整したい場合は以下に追加 * ================================================================= */ /* h3 { font-size: 1.5rem; } li { font-size: 1.1rem; } */ 💻 HTMLでの読み込み方 このCSSファイル（例としてファイル名を custom-sakura.css とします）を使用するには、HTMLファイルの セクションで、オリジナルの sakura.css の後に読み込む必要があります。 HTML Sakura CSS カスタム例
